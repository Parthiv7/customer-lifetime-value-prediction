# Customer Lifetime Value (CLTV) Prediction

## Overview

Customer Lifetime Value (CLTV) is a critical business metric that estimates the total revenue a customer is expected to generate throughout their relationship with a company.

This project develops a machine learning solution to predict CLTV using customer demographic information, policy details, residential attributes, and claim-related features. The objective is to help businesses identify high-value customers, optimize retention strategies, improve customer segmentation, and maximize long-term profitability.

---

## Problem Statement

The goal of this project is to build a machine learning model capable of predicting Customer Lifetime Value (CLTV) for unseen customers using historical customer and policy data.

Accurate CLTV prediction enables organizations to:

- Identify high-value customers
- Improve customer retention strategies
- Optimize marketing spend
- Personalize customer engagement
- Enhance long-term business profitability

---

## Dataset

The dataset consists of:

- **Train Dataset** – Historical customer records with CLTV values.
- **Test Dataset** – Customer records for which CLTV predictions are required.
- **Submission File** – Predicted CLTV values generated by the final model.

The available features include customer demographics, policy information, residential details, and claim-related attributes.

---

## Exploratory Data Analysis

Key observations from the dataset:

- Identified and removed **26 exact duplicate records** to improve data quality.
- Discovered **12,000+ records with identical feature values but different CLTV targets**, suggesting the presence of unobserved variables or external factors influencing customer lifetime value.
- The CLTV target variable exhibited a highly skewed distribution, making prediction challenging and reducing the suitability of R² as a standalone evaluation metric.
- Urban customers constituted the majority of the customer base.
- Most customers reported annual incomes in the **₹5–10 lakh** range.
- **Platinum A** emerged as the most frequently purchased policy type.
- The number of policies owned by a customer showed a strong positive relationship with CLTV.
- Residential area had a noticeable impact on customer value.
- Claim amount demonstrated a meaningful correlation with CLTV and emerged as an important predictive feature.

### Key Business Insight

The presence of a large number of records with identical customer and policy attributes but different CLTV values indicates that customer value is likely influenced by additional factors not captured in the dataset.

Potential missing variables include:

- Customer Age
- Employment Type
- Policy Pricing
- Customer Tenure
- Claim Frequency
- Behavioral Indicators

## Model Development

Several machine learning models were trained and evaluated:

- Linear Regression
- Random Forest Regressor
- XGBRF Regressor
- CatBoost Regressor
- Stacking Ensemble Models

### Model Comparison

| Model | Performance |
|---------|------------|
| Linear Regression | Baseline |
| Random Forest | Moderate |
| XGBRF Regressor | Strong |
| CatBoost Regressor | Strong |
| Stacking Ensemble | Slightly Better |

### Key Findings

- CatBoost and XGBRF consistently outperformed traditional regression models.
- Hyperparameter tuning resulted in only marginal performance improvements.
- Stacking ensembles provided a slight performance gain but reduced model interpretability.
- CatBoost offered the best balance between accuracy, robustness, and explainability.

---

## Final Model

### CatBoost Regressor

CatBoost was selected as the final model because it provided:

- Strong predictive performance
- Robust handling of categorical variables
- Minimal preprocessing requirements
- Stable validation results
- Better model interpretability through feature importance analysis

---

## Feature Importance

Feature importance analysis revealed that the following factors had the strongest influence on CLTV prediction:

- Number of Policies
- Claim Amount
- Income
- Residential Area
- Policy Type

These insights can assist businesses in identifying high-value customer segments and improving customer engagement strategies.
